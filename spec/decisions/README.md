## 運用ルール

- 1つの主要な判断につき1ファイルを作る
- ファイル名は `0001-topic-name.md` のように連番を付ける
- 状態は `検討中`、`採用`、`保留`、`却下`、`置換済み` のいずれかとする
- 採用後に判断が変わっても元の記録は消さず、新しい記録から参照する
- 比較時点の日付と、判断に使った公式情報へのリンクを残す
